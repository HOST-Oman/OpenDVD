ProjectLibre   1.6
Microsoft Project replacement

ProjectLibre is a project management software system. It intends to be a complete desktop replacement for Microsoft Project. ProjectLibre runs on the Java Platform, allowing it to run on the Linux, Mac OS or MS Windows operating systems.

Features:
  • Microsoft Project replacement
  • Gantt Chart
  • New and updated version of OpenProj with hundreds of updates/bug fixes
  • Earned Value Costing
  • PERT Charts
  • Project Management software
  • RBS Chart
  • New version of OpenProj
