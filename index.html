About

OpenDVD project provides a collection of high quality Open Source Software for Windows. Free and Open Source Software (FOSS) is provided with a license that gives the end user the right to use it freely for private or commercial use. You also have the right to inspect

and even modify the underlying source code. You can give away or sell the original version you received or one with your modification, though you may be required to in turn pass on the modified source code so that others can benefit from the changes you have made.
