Avidemux  2.6.9
Comprehensive video file editing

Avidemux is a video editor designed for simple cutting, filtering and encoding tasks. It natively supports a great number of file types, such as AVI, MPEG, VOB, TS, MP4, ASF, OGM, MKV and FLV, and natively supports a wide range of Video/Audio formats including MPEG-1, MPEG-2, MPEG-4 ASP, H.264/AVC, DV, HuffYUV, MP3, AAC, AC-3 and Vorbis.

Repetitive tasks can be automated using projects, job queue and powerful scripting capabilities. Video-DVDs and (S)VCDs can be created with easy-to-use "Auto" wizards.
